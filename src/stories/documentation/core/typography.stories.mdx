### Font family

<p>
  {process.env.STORYBOOK_APP_NAME} was designed with the{' '}
  <a target="_blank" href="https://fonts.google.com/specimen/Montserrat">
    Montserrat
  </a>{' '}
  font in mind. So be sure to follow these instructions. For instance, via
  Google Web Fonts:
</p>

```jsx
<link
  rel="stylesheet"
  href="https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700&display=swap"
/>
```

<Title>Box</Title>
<Subtitle>
  The Box component serves as a wrapper component for most of the CSS utility
  needs.
</Subtitle>

### Import

```jsx
import Box from 'tokenomy-ui/layout/Box';
```

### Usage

```jsx
<Box component="span" m={1}>
  <Button />
</Box>
```

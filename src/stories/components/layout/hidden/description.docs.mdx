<Title>Hidden</Title>
<Subtitle>
  Quickly and responsively toggle the visibility value of components and more
  with the hidden utilities.
</Subtitle>

### How it works

<p>
  Hidden works with a range of breakpoints e.g. <code>xsUp</code> or{' '}
  <code>mdDown</code>, or one or more breakpoints e.g. <code>only='sm'</code> or{' '}
  <code>only={['md', 'xl']}</code>. Ranges and individual breakpoints can be
  used simultaneously to achieve very customized behavior. The ranges are
  inclusive of the specified breakpoints.
</p>

### Import

```jsx
import Hidden from 'tokenomy-ui/layout/Hidden';
```

### Usage

```jsx
<Hidden xsUp>...</Hidden>
```

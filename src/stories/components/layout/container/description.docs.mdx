<Title>Container</Title>
<Subtitle>
  The container centers your content horizontally. It's the most basic layout
  element.
</Subtitle>

### Import

```jsx
import Container from 'tokenomy-ui/inputs/Container';
```

### Usage

```jsx
<Container>...</Container>
```

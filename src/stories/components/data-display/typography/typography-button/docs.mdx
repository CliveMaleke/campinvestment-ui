<Title>Button</Title>
<Subtitle>
  Button text is used to define the font that matches the button character.
</Subtitle>

### Import

```jsx
import Typography from 'tokenomy-ui/data-display/Typography';
```

### Usage

<TypographyButton
  type="semibold"
  buttonScale="small"
  style={{ marginTop: 20 }}
/>

```jsx
<Typography variant="button" type="semibold" buttonScale="small">
  Button
</Typography>
```

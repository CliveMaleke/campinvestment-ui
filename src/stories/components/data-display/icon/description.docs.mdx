<Title>Icons</Title>
<Subtitle>Guidance and suggestions for using icons with Tokenomy UI.</Subtitle>

### Import

```jsx
import Icon from 'tokenomy-ui/data-display/Icon';
```

### Usage

```jsx
<Icon>
  <path ... />  // SVG path
</Icon>
```

<Title>Skeleton</Title>
<Subtitle>
  Display a placeholder preview of your content before the data gets loaded to
  reduce load-time frustration.
</Subtitle>

### Import

```jsx
import Skeleton from 'tokenomy-ui/feedback/Skeleton';
```

### Usage

```jsx
<Skeleton />
```

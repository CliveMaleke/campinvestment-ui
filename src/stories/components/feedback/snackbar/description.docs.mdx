<Title>Snackbar</Title>
<Subtitle>
  Snackbars provide brief messages about app processes. The component is also
  known as a toast.
</Subtitle>

### Import

```jsx
import Box from 'tokenomy-ui/Feedback/Snackbar';
```

### Usage

```jsx
<Alert severity="general">...</Alert>
```

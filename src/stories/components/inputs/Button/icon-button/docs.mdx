<Title>Icon Button</Title>
<Subtitle>
  Buttons allow users to take actions, and make choices, with a single tap.
</Subtitle>

### Import

```jsx
import IconButton from 'tokenomy-ui/inputs/IconButton';
import DeleteIcon from 'tokenomy-ui/data-display/Icon';
```

### Usage

```jsx
<IconButton>
  <DeleteIcon />
</IconButton>
```
